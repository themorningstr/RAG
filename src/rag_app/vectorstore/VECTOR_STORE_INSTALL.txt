================================================================================
  DETAILED CONFIGURATION
================================================================================

CHROMA (Default - Already Works!)
  ────────────────────────────────
  VECTOR_STORE_TYPE=chroma
  # That's it! No other settings needed.
  # Data stored in: ./data/vector_store/
  
FAISS (Fast Local Search)
  ────────────────────────────────
  pip install faiss-cpu
  
  VECTOR_STORE_TYPE=faiss
  VECTOR_STORE_PATH=./data/vector_store
  
WEAVIATE (Docker Self-Hosted)
  ────────────────────────────────
  pip install weaviate-client
  
  docker run -d -p 8080:8080 semitechnologies/weaviate:latest
  
  VECTOR_STORE_TYPE=weaviate
  WEAVIATE_URL=http://localhost:8080
  
QDRANT (Docker Self-Hosted)
  ────────────────────────────────
  pip install qdrant-client
  
  docker run -d -p 6333:6333 qdrant/qdrant:latest
  
  VECTOR_STORE_TYPE=qdrant
  QDRANT_URL=http://localhost:6333
  
PINECONE (Cloud - Managed)
  ────────────────────────────────
  pip install pinecone-client
  
  1. Sign up: https://www.pinecone.io
  2. Create API key
  3. Create index "documents" with dimension 384
  
  VECTOR_STORE_TYPE=pinecone
  PINECONE_API_KEY=your-key-here
  PINECONE_ENVIRONMENT=us-west1-gcp
  
ASTRA DB (Enterprise)
  ────────────────────────────────
  pip install astrapy
  
  1. Sign up: https://astra.datastax.com
  2. Create serverless database
  3. Create API token
  
  VECTOR_STORE_TYPE=astradb
  ASTRA_DB_ID=your-id
  ASTRA_DB_REGION=us-east1
  ASTRA_DB_TOKEN=your-token

================================================================================
  FREQUENTLY ASKED QUESTIONS
================================================================================

Q: Which backend should I start with?
A: Chroma (default). It works immediately, no setup needed.

Q: Do I need to change my code to switch backends?
A: No! Same code works for all backends.

Q: How fast is the search?
A: FAISS is fastest (10ms), others are 30-100ms for 100k documents.

Q: What's the cost?
A: Chroma, FAISS, Weaviate, Qdrant are free.
   Pinecone starts at $0.60/million vectors.
   Astra DB is usage-based.

Q: Can I use multiple backends at once?
A: Yes! Each stores data separately.

Q: How do I back up my data?
A: Chroma/FAISS use files. Cloud services have built-in backups.

Q: What if I change my mind?
A: Switch anytime! Just edit .env and reingest.

Q: Do I need Docker for self-hosted?
A: For Weaviate/Qdrant yes, but optional.
   Chroma/FAISS need no external services.

Q: Which is best for production?
A: Pinecone or Astra DB (fully managed, no ops).

Q: Which is best for cost?
A: Chroma, FAISS, or self-hosted Qdrant (free).
